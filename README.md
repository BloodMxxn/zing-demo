# زینگ (Zing)

سیستم مدیریت گزارش‌های فروشگاه — گزارش‌دهی شیفت‌ها به جای پیام‌رسانی.

## Tech Stack

- React 19 + TypeScript
- Vite 8
- Tailwind CSS 4
- Radix UI
- Lucide React
- Zustand
- React Router DOM 7

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Type check + production build |
| `npm run lint` | Run ESLint |
| `npm run preview` | Preview production build |
